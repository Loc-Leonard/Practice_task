# dembo-rubinstein
## структура проекта
```
dembo-rubinstein-app/
├── main.go                 # Основной сервер на Go
├── go.mod                  # Модуль Go
├── static/                 # Статические файлы
│   ├── css/
│   │   └── style.css       # Стили для вертикальных шкал
│   ├── js/
│   │   ├── survey.js       # Логика опроса
│   │   └── results.js      # Логика результатов (для варианта А)
│   └── index.html          # Главная страница с опросом
├── templates/              # HTML шаблоны Go (для варианта Б)
│   └── result.html         # Шаблон результатов
└── data/                   # Для хранения (опционально)
    └── store.go            # Логика хранения данных
```
